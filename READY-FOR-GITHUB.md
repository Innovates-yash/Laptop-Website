## 🚀 How to Push (Quick Version)

### Step 1: Install Git
If you don't have Git:
- Download: https://git-scm.com/download/win
- Install with default settings
- Restart Command Prompt

### Step 2: Open Command Prompt
- Navigate to `voltex` folder
- Click address bar → type `cmd` → Enter

### Step 3: Run These Commands
```bash
git init
git add .
git commit -m "Initial commit: VOLTEX e-commerce platform with 3D animations"
git remote add origin https://github.com/Innovates-yash/Laptop-Website.git
git branch -M main
git push -u origin main
```

### Step 4: Authenticate
When prompted:
- **Username:** Innovates-yash
- **Password:** Your Personal Access Token (get from GitHub Settings → Developer settings → Personal access tokens)
